# Guildmasters Roadmap

## v0.1 - First Playable

Goal: prove the basic idle loop works.

Required:

- Save state foundation
- Guild state
- Gold
- Reputation
- Basic hero roster
- Hero recruitment
- Contract board
- Contract assignment
- Contract completion timer
- Visible success chance
- Success reward
- Failure reward

Done when:

- A player can recruit at least one hero.
- A player can send that hero on a contract.
- The contract resolves after time passes.
- The player receives rewards.
- The state persists after reload.

## v0.2 - Progression

Goal: make repeated play meaningful.

Required:

- Hero leveling
- Guild leveling
- Better recruits by guild level
- Better contracts by reputation
- Basic upgrade costs

## v0.3 - Offline Progress

Goal: make the game function as a real idle game.

Required:

- Last-seen timestamp
- Contract completion while away
- Offline reward collection
- Safe time repair for malformed saves

## v0.4 - Contract Expansion

Goal: add meaningful risk tiers without bloating systems.

Required:

- Common contracts
- Elite contracts
- Legendary contracts
- Contract difficulty scaling
- Success chance bounds
- Partial rewards on failure

## v0.5 - Records and Polish

Goal: give the guild permanence.

Required:

- Lifetime gold earned
- Contracts completed
- Failed contracts
- Legendary clears
- Heroes recruited
- Highest guild level

## v1.0 - Complete Core Game

Goal: finish the simple idle guild loop.

Required:

- Stable save system
- Complete recruit -> contract -> reward -> upgrade loop
- Meaningful guild progression
- Offline progress
- Contract risk and failure
- Records screen
- Mobile-readable UI
